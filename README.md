# 《Unicode查询》

基于 `unicode查询.html` 增强而来的单文件 HTML 工具，纯前端、离线可用，开箱即用。

---

## 相比原版新增了什么

### 1. 随机摇字 — 语言文字分区

随机摇字的分区下拉框最前面新增了一组「按语言文字」选项，选哪个语言就只在那个语言的 Unicode 范围内摇字。

目前支持的语言/符号分区共 20 个：

| 分区 | 覆盖范围 |
|------|----------|
| 中文 | CJK 统一表意文字（基本区 + 扩展 A–H）、兼容表意文字、康熙部首、笔画、注音符号、CJK 标点 |
| 英文（拉丁字母） | 基本 ASCII、拉丁补充、拉丁扩展 A/B、拉丁扩展附加 |
| 日文 | 平假名、片假名、CJK 标点、汉字、半角全角形式 |
| 韩文 | 谚文音节、谚文字母、谚文兼容字母、半角全角形式 |
| 希腊文 | 希腊字母与科普特字母、希腊扩展 |
| 西里尔文（俄文等） | 西里尔字母、西里尔补充 |
| 阿拉伯文 | 阿拉伯字母、扩展、演示形式 |
| 希伯来文 | 希伯来字母、演示形式 |
| 泰文 | 泰文字母 |
| 天城文（印地文） | 天城文字母 |
| 孟加拉文 | 孟加拉文字母 |
| 藏文 | 藏文字母 |
| 数学符号 | 数学运算符、杂项数学符号、高级数学运算符 |
| 箭头符号 | 箭头、补充箭头 A/B、杂项箭头 |
| 几何图形 | 几何图形 |
| 制表符 | 制表符、方块元素 |
| 标点符号 | 一般标点、补充标点 |
| 货币符号 | 货币符号 |
| 杂项符号 | 杂项符号、印刷符号 |
| 表情符号 | 表情符号、杂项符号与象形文字、运输与地图、补充符号与象形文字等 |

### 2. 收藏 — 按语言分组

收藏视图原来有「按区块分组」和「平铺显示」两种模式，现在中间多了一个 **「🌐 按语言分组」** 按钮。点击后，收藏的字符会按所属语言/文字自动归类，分组按数量从多到少排列。

### 3. 详情弹窗 — 字符含义

点开任意字符的详情弹窗，在「基本信息」下方多了一个 **「💡 字符含义」** 区块。内置约 742 条含义数据，覆盖：

- ASCII 符号（`!` `@` `#` `$` `%` 等）
- Unicode 杂项符号（`★` `☀` `☂` `☮` `♻` 等）
- 数学运算符（`∞` `∑` `∫` `√` `≈` `≠` 等）
- 货币符号（`€` `¥` `₹` `₽` `₿` 等）
- 表情符号（`😀` `😂` `❤` `👍` `🔥` 等）
- 常见中文汉字（约 180 字，含一画至常用字）
- 希腊字母（Α α … Ω ω，含数学含义）
- 日文假名（あ い う … ア イ ウ … 全套）
- 韩文常见音节（가 나 다 … 하）

**查不到含义的字符会显示「无」。**

## ⚠️ 重要提醒：分区可能不准确

**语言文字分区基于 Unicode 码位区间划分，可能不准确。** 原因如下：

1. **区间重叠** — 部分字符可能同时属于多个语言（例如汉字同时被中文和日文使用，半角全角形式同时被日文和韩文使用）。工具按分区列表的先后顺序匹配，中文优先于日文，因此汉字会归入「中文」。这只是实现选择，不代表语言学结论。

2. **范围有限** — 分区只覆盖了常见的 Unicode 区块，未覆盖的字符会归入「其他」。某些稀有文字、历史文字、私人使用区字符无法被归类。

3. **分类非语言学标准** — 这里的「语言」实际是「文字系统（script）」的粗略对应，并非严格的语言学分类。例如「英文」实际指的是拉丁字母，「俄文」实际指的是西里尔字母。

4. **字符含义有限** — 含义字典只收录了约 742 条，绝大多数 Unicode 字符查不到含义，会显示「无」。含义为人工整理，可能有疏漏或不严谨之处。

**如果对分类精度有要求，请以 Unicode 官方标准（[Blocks](https://www.unicode.org/Public/UCD/latest/ucd/Blocks.txt)、[Scripts](https://www.unicode.org/Public/UCD/latest/ucd/Scripts.txt)）为准。**
